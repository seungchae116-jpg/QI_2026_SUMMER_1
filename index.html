이 예측은 겉모습 기반 숙성 단계 추정이며 실제 맛·당도를 측정하지 않아요. 학습 데이터는 스튜디오 조명 환경에서 촬영되어, 실제 주방 조명에서는 정확도가 달라질 수 있어요.

왜 Avomatic인가요

"지금 몇 단계인지"는 이미 다들 알려줘요. 우리는 세 가지가 달라요.

📅

상태가 아니라 계획에 답해요

"3단계입니다" 대신 "냉장 보관하면 토요일에 딱 맞아요"라고 말해요. 보관 온도별 숙성 속도 차이(10°C는 20°C보다 약 2.3배 느려요)를 논문 실측 계수로 환산해요.

🙋

보관법은 추측하지 않고 물어봐요

저장 조건은 사진만으로 알 수 없어요. 같은 단계라면 냉장이든 실온이든 겉모습이 같거든요. 이 값을 무시하면 "며칠 남음" 추정이 부정확해져요.

🔍

한계를 숨기지 않아요

학습 데이터는 스튜디오 조명에서 촬영됐고, 모델은 외관상 단계를 예측할 뿐 실제 "맛"을 측정하지 않아요. 결과 화면과 아래 섹션에 그대로 밝혀요.
